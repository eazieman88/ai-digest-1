# Задача
Напиши вечерний выпуск за сегодня в двух версиях:
1. Блог (600–900 слов) — полная версия MDX с frontmatter
2. Telegram (250–350 слов) — выжимка, ссылка на блог в конце

Включи: обзор дня (sentiment + что могло случиться но не случилось),
курсы ЦБ РФ (на сегодня и на завтра если опубликован), рынки (индексы,
commodities, крипта — только где было движение), главную новость дня
(разные мнения, альтернативные сценарии), прогноз на завтра (конкретные
уровни и события). «Что мне с этого?» — если тема заслуживает.

# Формат
- Блог: файл YYYY-MM-DD-evening.mdx
- Frontmatter: title, date, type: evening, tags, description
- Таблицы для курсов и ставок
- Telegram: отдельный блок после <!-- telegram -->
- В конце Telegram-версии: «Полный разбор в блоге →»

# Ограничения
- Без прямых рекомендаций «покупай/продавай»
- Блог: развёрнутая аналитика, цитаты, несбывшиеся сценарии
- Telegram: только суть — цифры, вывод, прогноз. Без развёрнутой аналитики
- Emoji только в заголовках рубрик
- Дисклеймер встроен в layout, не дублировать в блог-версии
- В Telegram-версии дисклеймер одной строкой в конце
