# Контекст
Финансовый дайджест для практикующих трейдеров.
Утренний выпуск перед открытием торговой сессии.
Публикация: блог (Astro) + Telegram-канал.

# Ссылки
Формат из CLAUDE.md, секция «Утренний пост».
Стиль и тон — из секции «Стиль и тон».
Различия блог vs Telegram — из секции «Различия блог vs Telegram».

# Задача
Напиши утренний выпуск за сегодня в двух версиях:
1. Блог (300–500 слов) — полная версия MDX с frontmatter
2. Telegram (150–200 слов) — сжатая версия, ссылка на блог в конце

Включи: обзор ночных событий, курсы ЦБ РФ (USD/RUB, EUR/RUB, EUR/USD),
ключевые уровни с пояснением логики, фокус дня.

# Формат
- Блог: файл YYYY-MM-DD-morning.mdx
- Frontmatter: title, date, type: morning, tags, description
- Таблица курсов в markdown
- Telegram: отдельный блок после <!-- telegram -->
- В конце Telegram-версии: «Подробнее в блоге →»

# Ограничения
- Без кликбейтных заголовков
- Не пересказывать новости — давать аналитику
- Термины без пояснений
- Emoji только в заголовках рубрик
- Дисклеймер встроен в layout, не дублировать в блог-версии
- В Telegram-версии дисклеймер одной строкой в конце
