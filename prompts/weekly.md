# Контекст
Финансовый дайджест для практикующих трейдеров.
Воскресный недельный обзор — главный выпуск недели.
Публикация: воскресенье 18:00 МСК, блог + Telegram.

# Ссылки

Источники: ATAS, CME, CFTC COT, TradingView, FRED,
ЦБ РФ, ForexFactory, Bloomberg, РБК.
Различия блог vs Telegram — из секции «Различия блог vs Telegram».

# Задача
Напиши воскресный обзор за прошедшую неделю в двух версиях:
1. Блог (800–1200 слов) — полная версия MDX с frontmatter
2. Telegram (400–500 слов) — сокращённая версия, ссылка на блог

Блог включает:
- Итоги недели — что двигало рынки, несбывшиеся сценарии
- Что показали объёмы — анализ через ATAS по ключевым инструментам
- Макро-блок — ставки ЦБ/ФРС/ЕЦБ, инфляция, безработица, PMI, состояние экономик РФ/US/EU
- Сетап недели — один инструмент с разбором через volume profile
- Мысль на неделю — ключевые события, на что смотреть
- Что мне с этого? — обязательно

Telegram включает:
- Итоги + сетап + мысль на неделю
- Без полного макро-блока и объёмного анализа

# Формат
- Блог: файл YYYY-MM-DD-weekly.mdx
- Frontmatter: title, date, type: weekly, tags, description
- Таблицы для курсов и ставок
- Telegram: отдельный блок после <!-- telegram -->
- В конце Telegram-версии: «Полный обзор с графиками в блоге →»

# Ограничения
- Без прямых рекомендаций «покупай/продавай»
- Макро-блок — как данные влияют на рынки, не пересказ
- Сетап — образовательный разбор, не сигнал
- Emoji только в заголовках рубрик
- Дисклеймер встроен в layout, не дублировать в блог-версии
- В Telegram-версии дисклеймер одной строкой в конце
