# Контекст
Финансовый дайджест для практикующих трейдеров.
Вечерний выпуск в разговорном стиле.
Публикация: блог (Astro) + Telegram-канал.

# Ссылки
Формат из CLAUDE.md, секция «Вечерний пост».
Различия блог vs Telegram — из секции «Различия блог vs Telegram».

# Задача
Напиши вечерний выпуск за сегодня в двух версиях:
1. Блог (600–900 слов) — полная версия MDX с frontmatter
2. Telegram (250–350 слов) — выжимка, ссылка на блог в конце

Стиль: разговорный. От первого лица или обращение к читателю.
Живой тон, короткие предложения.
«Рынок развернулся за одну ночь. Вчера росли на надеждах мира —
сегодня всё обнулилось. Следите за нефтью у $110, это сейчас
главный уровень.»

# Формат
- Блог: файл в src/content/blog/, MDX с frontmatter
- Frontmatter: title, date, type: evening, tags, description
- Telegram: отдельный блок после <!-- telegram -->

# Ограничения
- Разговорный тон, но без панибратства и сленга
- Можно обращаться к читателю на «вы»
- Без прямых рекомендаций «покупай/продавай»
- Emoji только в заголовках рубрик
- Дисклеймер в layout, не дублировать
