## 5. Hero-баннер Old Money v2 — `hero-old-money-v2.svg`

Создай SVG hero-баннер 1200x630 для Telegram-канала «Биржевой Дэнди» (Bourse Dandy).

Стиль: Old Money / Private Banking. НЕ Art Deco, НЕ гламур.
Настроение: кабинет управляющего хедж-фондом в Лондоне. Сдержанная роскошь.

Палитра:
- Фон: глубокий hunter green (#162216 → #0e1a0e), градиент вертикальный
- Золото: состаренное, приглушённое (#c9a94e → #8b7332), НЕ яркое блестящее
- Акцент: кремовый для вторичного текста (#d4c9a8)
- Свечи: приглушённый зелёный (#5a8a5a) и бордовый (#8a5a5a), НЕ яркий красный/зелёный

Композиция:
- Левая треть: силуэт мужчины в цилиндре, с тростью, монокль с цепочкой. Силуэт почти чёрный (#0a120a), как тень в дверном проёме. Аскотный галстук, жилет с цепочкой карманных часов — gold accents, opacity 0.3-0.5. Стрелки на брюках, оксфордские туфли.
- Правая часть: текст
  - Мелким шрифтом сверху: «FINANCIAL DIGEST» (caps, letter-spacing 6, opacity 0.35)
  - Заголовок: «Биржевой» + «Дэнди» — Georgia italic bold, 58px, приглушённое золото
  - Разделитель: тонкая двойная линия с точкой по центру
  - Под разделителем: «BOURSE DANDY» (caps, letter-spacing 5, opacity 0.35)
  - Тагрлайн: «Расчёт. Позиция. Результат.» — italic, 18px, кремовый, opacity 0.45
  - Теги: «Order Flow · Volume Profile · Макроаналитика» + «Акции · Фьючерсы · Крипта · Forex · Commodities» — Georgia 12px, opacity 0.2
  - Внизу: «EST. MMXXVI» — caps, 10px, letter-spacing 3, opacity 0.2

Фоновые детали:
- Пинстрайп-текстура по всему фону (вертикальные линии, шаг 8px, gold, opacity 0.06)
- Тонкая золотая рамка (двойная, внешняя 1px, внутренняя 0.4px, opacity 0.35/0.2)
- Угловая филигрань — скруглённые уголки с точкой (не Art Deco, а как на банкноте)
- Свечной график на правом фоне — еле видимый (opacity 0.12-0.15), приглушённые цвета, горизонтальная сетка

Чего НЕ делать:
- Яркие неоновые зелёный/красный
- Art Deco ромбы и острые углы
- Тени с сильным blur
- Слишком много деталей в силуэте — он должен читаться как тень

## 6. Hero-баннер Old Money v3 — `hero-old-money-v3.svg`

На базе промпта v2. Отличия:

Сделай черты силуэта более яркими — трость, монокль, лента на шляпе, детали пиджака. Конкретно:

- Лента на шляпе: opacity 0.8 + нижняя линия-край
- Монокль: stroke 2.5px, opacity 0.85 + блик на стекле (path, opacity 0.35)
- Цепочка монокля: stroke 1.2, opacity 0.6
- Лацканы пиджака: stroke 1.5, opacity 0.5 + линии бортов (stroke 0.8, opacity 0.25) + нижний край пиджака (opacity 0.3)
- Трость — ручка: stroke 5, opacity 0.7 + набалдашник (circle r=5, opacity 0.5) + кольцо-воротник
- Трость — наконечник: stroke 5, opacity 0.6
- Трость — блик на древке: золотая тонкая линия (stroke 0.6, opacity 0.2)
- Карманные часы: цепочка (stroke 1, opacity 0.45) + корпус (circle r=3, opacity 0.35)
- Платок в кармане: opacity 0.45
- Пуговицы жилета: opacity 0.35
- Стрелки на брюках: gold stroke 0.6, opacity 0.2
- Обувь: линия-деталь opacity 0.18

Добавить отдельный градиент `gldBright` (#d4b85a → #c9a94e → #d4b85a) для ярких элементов (монокль, ручка трости, лента, лацканы).

Силуэт остаётся тенью (#0a120a), но золотые детали читаются чётко.

## 7. Логотип Old Money v2 — `logo-old-money-v2.svg`

Создай SVG логотип 512x512 для канала «Биржевой Дэнди» (Bourse Dandy).

Стиль: фамильный герб / эмблема частного клуба. Old Money, НЕ гламур.

Палитра:
- Фон за щитом: чёрно-зелёный (#0e1a0e)
- Щит: hunter green (#1b2a1b) с золотой обводкой (#c9a94e, 2-3px)
- Золото: состаренное, неоднородное (градиент #c9a94e → #8b7332 → #c9a94e)

Структура:
- Форма: геральдический щит (заострённый снизу)
- Внутри щита двойная обводка (внешняя 3px, внутренняя 1px, opacity 0.4)
- Верхняя половина: силуэт цилиндра (тёмный, с золотой обводкой и лентой)
  - Лента на цилиндре — тонкая, одна линия по центру, золото opacity 0.6
  - Цилиндр должен быть читаем даже при 40x40 px
- Горизонтальный разделитель (тонкая линия, opacity 0.3)
- Нижняя половина: монограмма «БД» — переплетённая, italic serif bold
  - Буквы перекрываются как в классических монограммах (Б слева, Д справа, с общей вертикалью)
  - Золото, opacity 0.85
- Под монограммой: маленькая точка-разделитель
- «BOURSE DANDY» — 13px, caps, letter-spacing 4, opacity 0.45
- Внизу щита: «EST. MMXXVI» — 10px, caps, letter-spacing 3, opacity 0.3

Внешние элементы:
- Лавровые ветви по бокам щита — тонкие, стилизованные, gold stroke 1.2px, opacity 0.35
- Листья как заливка с opacity 0.15
- За монограммой (внутри щита): два перекрёстных графика (бычий и медвежий) — очень subtle, opacity 0.15

Детали выделены ярче чем в v1: обводка цилиндра 2px, лента 0.8 opacity, монограмма кириллицей «БД» 95px.

При масштабировании до 40x40:
- Должны читаться: щит, цилиндр, монограмма
- Остальное может пропадать

Чего НЕ делать:
- Круглая форма (это герб, не монета)
- Art Deco элементы
- Яркие цвета свечей
- Мелкий нечитаемый текст внутри щита
